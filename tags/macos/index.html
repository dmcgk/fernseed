This is phenomenally niche, but if you use full-screen apps regularly in macOS and are driven fully insane by one of those windows randomly stealing focus when you switch to one of the others:

defaults write com.apple.Dock workspaces-auto-swoosh -bool NO

(Then kill and restart the Dock)
