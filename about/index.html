Automation

  • just publish_refresh: refresh Mastodon content, rebuild, copy to ../fernseed, and push with a site refresh YYYY-MM-DD commit message.
  • just publish_build: rebuild/copy and push to ../fernseed with a site build YYYY-MM-DD commit message.
  • just auto_refresh_install 1800: install a launchd agent to run periodic refresh/publish every 1800 seconds.
  • just auto_refresh_status: show launchd job status.
  • just auto_refresh_uninstall: remove the launchd job.
